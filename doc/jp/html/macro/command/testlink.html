testlink

現在のリンクおよび接続の状態を報告する。 

testlink

解説

リンクと接続の状態に応じて以下のうちの一つの値がシステム変数 result に格納される。

値 状態
0 Tera Term とリンクされていない。
1 ホストへの接続はされていないが、Tera Term へのリンクはされている。
2 リンクおよび接続両方ともされている。

例

; もし MACRO が Tera Term にリンクされていなければ
testlink
; connect コマンドを実行
if result=0 connect 'host'
