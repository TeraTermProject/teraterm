Proxy（[Setup] メニュー）

"Proxy Setup" ダイアログボックス

Type
プロキシのタイプを設定します。
  • Winthout Proxy：プロキシを使用しません。
  • HTTP：HTTP プロキシを使用します。
  • TELNET：TELNET プロキシを使用します。
  • SOCKS4：SOCKS4 プロキシを使用します。
  • SOCKS5：SOCKS5 プロキシを使用します。
Hostname
プロキシサーバを指定します。
Port number
プロキシサーバのポート番号を指定します。
Username
プロキシサーバの利用に認証が必要な場合、そのユーザ名を指定します。
Passwd
プロキシサーバの利用に認証が必要な場合、そのパスワードを指定します。
Other setup
"Other Setup" メニューに移ります。

"Other Setup" メニュー

Common setup

Connection Timeout
プロキシとの通信（接続維持用の通信を含む。） がここで指定した秒数だけ途絶えたら接続を切ります。
Proxy connection log
プロキシとの通信を指定したファイルに記録します。

SOCKS proxy setup

Name で名前解決の方法を指定します。

auto resolve
自動
resolve remote
SOCKS サーバ側で名前解決します。
resolve local
Tera Term が動いている PC で名前解決します。

TELNET Proxy setup

TELNET プロキシに関する設定を行います。

Hostname prompt
ホスト名の入力を催すプロキシ側のプロンプトを指定します。
Username prompt
ユーザ名の入力を催すプロキシ側のプロンプトを指定します。
Passwd
パスワードの入力を催すプロキシ側のプロンプトを指定します。
Connected
接続完了のメッセージを指定します。
Error message
プロキシ側からのエラーメッセージの表示を指定します。
