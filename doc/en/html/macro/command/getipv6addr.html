getipv6addr

Retrieves local IPv6 address(es) 

getipv6addr <string array>

Parameters

Array <string array>
IP addresses are stored in the string array.

Return Value

System variable <result>
It is set to the IP address number.
If the IP address is nothing, it is set to 0.
When the IP address can not retrieve, it is set to -1. It is always failed on Windows 2000 or earlier.

Example

strdim ipaddr 10

getipv6addr ipaddr
if result = -1 then
	messagebox 'Error' 'IPv6 address'
elseif result = 0 then
	messagebox 'Not found' 'IPv6 address'
else
	N = result - 1
	for i 0 N
		messagebox ipaddr[i] "IPv6 address"
	next
endif
