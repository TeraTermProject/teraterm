getipv6addr

Retrieves local IPv6 address(es) 

getipv6addr <string array>

Parameters

Array <string array>
IP addresses are stored in the string array.

Return Value

System variable <result>
It is set to the IP address number.
If the IP address is nothing, it is set to 0.

Example

strdim ipaddr 10

getipv6addr ipaddr
N = result - 1
for i 0 N
	messagebox ipaddr[i] "IPv6 address"
next
