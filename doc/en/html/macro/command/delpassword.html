delpassword

Deletes a password. 

delpassword <filename> <password name>

Parameters

string <filename>
Password file name.
string <password name>
Password identifier.

Remarks

Deletes a password specified by <password name> in the password file <filename>.
If <password name> is a blank string, all passwords in the file are deleted.

Example

delpassword 'password.dat' 'foo'
