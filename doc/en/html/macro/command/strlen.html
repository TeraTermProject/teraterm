strlen

Returns a length of string.

Format

strlen <string>

Remarks

Returns the length of <string> in the system variable "result".
The length does not include the null-terminate character. Also, the length counts in terms of bytes.

Example

strlen Txcnt
if result>0 then
	logwrite Txcnt
else
	messagebox 'Txcnt is NULL' 'error'
endif
