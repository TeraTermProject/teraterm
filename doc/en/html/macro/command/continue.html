continue

Transfers control immediately to the next iteration of a loop. 

continue

Remarks

Transfer from inside a "for" or "while" loop to the next iteration of that loop.

Example

a=0
b=0
while a<5 
    a=a+1
    if a>2 then
        continue
    endif
    b=b+1
endwhile

sprintf2 var "a=%d b=%d" a b
dispstr var
;a=5 b=2

Reference

do, loop
for, next
until, enduntil
while, endwhile
