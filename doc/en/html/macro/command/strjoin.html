strjoin

Concatenates a string with delimiters.

Format

strjoin <strvar> <separator> <count> [<special>]

Remarks

Concatenates a specified separator string variable <strvar> between each element of the 9 system variables from "groupmatchstr1" to "groupmatchstr9", yielding a single concatenated string.

The <count> is specified the maximum number of substrings to concatenate.

If <special> is non-zero, following strings in <message> is treated as spacial character.
If <special> is not specified then 1 is used.

input output
\\ "\"
\n line feed
\t tab

Example

groupmatchstr1="Jan"
groupmatchstr2="Feb"
groupmatchstr3="Mar"
groupmatchstr4="Apr"
groupmatchstr5="May"
groupmatchstr6="Jun"
groupmatchstr7="Jul"
groupmatchstr8="Aug"
groupmatchstr9="Sep"
strjoin strvar " and " 6
messagebox strvar "strvar"   ; 'Jan and Feb and Mar and Apr and May and Jun'

groupmatchstr1="Jan"
groupmatchstr2="Feb"
groupmatchstr3="Mar"
groupmatchstr4="Apr"
groupmatchstr5="May"
groupmatchstr6="Jun"
groupmatchstr7=""
groupmatchstr8=""
groupmatchstr9=""
strjoin strvar "," 9
messagebox strvar "strvar"   ; 'Jan,Feb,Mar,Apr,May,Jun,,,'
