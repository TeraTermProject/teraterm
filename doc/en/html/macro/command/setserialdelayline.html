setserialdelayline

Sets the time intervals between lines being sent by Tera Term. (version 5.5.0 or later)

Format

setserialdelayline <delay>

Parameters

integer <delay>
The time intervals between lines. Unit: milliseconds.

Remarks

Changes the time interval between lines being sent by Tera Term, as specified by <delay> [ms].

Example

setserialdelayline 50
