TTXttyplay command line

You can use the following command line parameters on TTXttyplay.

Options

/ttyplay
/TP (short option for /ttyplay)
Enable TTXttyplay. Replay the file that specified by /R= option as ttyrec format.
/TP=1
Enable TTXttyplay. Replay the file that specified by /R= option as ttyrec format.
Display to recording time at title bar.
/TP=3
Enable TTXttyplay. Replay the file that specified by /R= option as ttyrec format.
Display to recording time at title bar. Include milliseconds.
/ttyplay-nowait
/tpnw (short option for /ttyplay-nowait)
no-wait mode.
/TPW=[<MaxWait>][,[<MinWait>]]
wait period limit
  • MaxWait
    maximum wait period[seconds]
    If the wait period exceeds MaxWait, over period be cut off.
  • MinWait
    minimum wait period[seconds]
    If the wait period is less than MinWait, that wait will be skipped.
/TPN
It displays only filename in the title bar path. (same /TPN=1)
/TPN=<count#>
Number of path segment of displayed in the title bar
  • /TPN=0 full path
  • /TPN=1 filename
  • /TPN=2 parent directory name and filename
If count# is set to 3 or grater, multiple parent directory name and filename are displayed.

Example

ttermpro.exe /ttyplay /R="C:\Documents and Settings\doda\My Documents\record.tty"
ttermpro.exe /ttyplay /ttyplay-nowait /R=example.tty
ttermpro.exe /ttyplay /TPW=5,2 /R=example.tty
