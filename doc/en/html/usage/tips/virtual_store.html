Virtual Store

File Virtualization on Vista later

When a user writes the data to special file and registry on the Windows Vista later,
the data will be automatically redirected to another path. This feature is the "Virtual Store" called.
The feature is enabled when the User Account Control(UAC) function of the operating system is enabled
and Tera Term is not executed as the Administrator privilege.

When the "Virtual Store" function is enabled, the file stored into the "%ProgramFiles(x86)%\teraterm"
path will be redirected to the "%LOCALAPPDATA%\VirtualStore\Program Files (x86)\teraterm" path.
Target file are teraterm.ini, log file and ssh_known_hosts file.
