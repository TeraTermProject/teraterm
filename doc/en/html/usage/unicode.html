Unicode Settings

Additional settings / "Encoding" tab for setting character encoding. To use UTF-8, select the following character encoding.
Chinese/UTF-8
Japanese/UTF-8
Korean/UTF-8
Select UTF-8.
By default, Ambiguous Characters width is 2 and Emoji Characters width is 2 for CJK.
UTF-8
Select UTF-8.
By default, Ambiguous Characters width is 1 and Emoji Characters width is 1.

command line

When "UTF8" is specified with '/KT' or '/KR' option in the command line, UTF-8 encoding/decoding can be used during transmitting and receiving of data.
