Additional settings / "Plugin" tab ([Setup] menu)

Set enable(load)/disable(unload) plugins.

Display list of plugins in the plugin folder and settings.

Plugins are loaded at startup. After startup, plugins cannot be loaded and unloaded. Selecting Restore Settings will not reload or unload plugin.

path
plugin path
enable
You can change it by right-clicking.
Settings take effect at next startup.
enable
Load plugin
disable
Do not load the plugin.
unspecified
enable/disable setting has not been configured. Default state is unspecified.
unspecified plugins is loaded.
loaded
show plugin is currently loaded.
loaded
Plugin is loaded.
unloaded
Plugin is not loaded.
order
The priority of plugins.
