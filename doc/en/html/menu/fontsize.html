[FontSize] menu

The registered font size list are shown on the menu bar.
When the entry is selected, the font size of Tera Term window is automatically changed.

Increase
Increase the font size.
Decrease
Decrease the font size.

NOTICE

  • This feature can be only used by using the TTXChangeFontSize plugin.
  • When no one menu entry is registered, the [FontSize] menu can not be displayed. A user must register the menu entry.
