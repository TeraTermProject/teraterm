Additional settings / "Theme" tab ([Setup] menu)

Theme

Looks of Tera Term can be changed.
It is not a Windows theme (dark theme, etc.) setting.

Enable
The theme will be enabled.
Theme Editor
Open Theme editor.
Disable theme when moving or resizing window
Faster drawing by omitting drawing of the window background (BG theme) during window resizing, moving, and Z-order changes.
This can be used when using the BG theme.
Startup
Sets the theme to be used at startup.
no use
No theme is used
Fixed theme file
Loads a specific theme file
Random Theme file
Randomly load themes from theme files folder
Susie Plug-in path:
Path where Susie Plug-in files are stored.

Use Plug-in with same architecture as Tera Term binary. The architecture is determined by the Plug-in's file extension.

Plug-in file extension
architecture extension
x86 32bit .spi
x86 64bit .sph
ARM 64bit .spha
