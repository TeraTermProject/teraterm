Additional settings / "UI" Tab ([Setup] Menu)

UI Language
The message catalog(menu and dialog message) can be changed by specifying the language file(lang_utf16le\*.lng). Refer to Language file.
Dialog Font
Specify Dialog font.

If you set large size or wide font for the dialog font, characters may be cut off, or dialog may be missing characters or dialog may extend beyond the screen, making it impossible to press [ok] button.

Select Button
Opens font dialog to select font to use.
Fonts displayed in the font dialog list can be modified using the checkboxes below.
Default button
Discards current font setting, and selects the default dialog font used by Windows.
List proportional fonts in font dialog
Proportional fonts are displayed in font list in font dialog. If the currently selected font is a proportional font, this checkbox is automatically checked and cannot be changed.
This checkbox changes behavior of font dialog and is not saved in settings file.
List hidden fonts in font dialog
Displays fonts set as hidden by the OS in font list in font dialog.
If the currently selected font is a hidden font, this checkbox is automatically checked and cannot be changed.
This checkbox changes behavior of font dialog and is not saved in settings file.

This feature to hide certain fonts in font dialog is available on Windows 7 and later.

Click Font Folder link to open font folder where you can preview fonts and show/hide settings.
