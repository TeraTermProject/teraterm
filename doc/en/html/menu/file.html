[File] menu

New connection...
Connects to a new host or a serial port or a named pipe.
Duplicate session
Duplicates current session. Only TELNET, SSH and Cygwin connection is duplicated.
Please refer to Notes on Specifying a Password of TTSSH command line.
Log...
Logs received characters to a file.
Pause Logging
Pause logging.
Comment to Log...
Adds comment to the log file. Maximum allowed length of comment is 1024Byte.
(the number of characters that can be used in a comment varies depending on the character encoding.)
View Log
Opens the log file.
Show Log dialog...
Displays the log dialog.
Stop Logging
Stop logging and closes the log file.
Send file...
Sends a file to the host.
When this menu is selected
Open "Send file" dialog.
When this menu is selected with holding down CTRL key
Open file select dialog, "Send file" dialog is skipped.
Previously used send options are used.
behavior can be swapped by setting.
Transfer
Transfers files between the PC and the host with the Kermit/XMODEM/YMODEM/ZMODEM/B-Plus/Quick-VAN protocols.
SSH SCP
Transfers files between the PC and the host with the Secure Protocol.
Change directory...
Changes the current directory for file transfers.
Replay Log...
Replays the binary log file. This feature and the /R option are equivalent.
Print...
Prints the current screen, or the selected text.
Disconnect
Closes the connection. If the connection type is TCP/IP and the Auto window close option is selected, Tera Term is also closed.
Exit
Closes Tera Term. If the connection is open, it is also closed.
Exit All
Closes All Tera Term instances without confirmation. If the connection is open, it is also closed.
