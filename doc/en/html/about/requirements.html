Requirements

Supported Windows versions:

  • Microsoft Windows 7
  • Microsoft Windows 8
  • Microsoft Windows Server 2012
  • Microsoft Windows 8.1
  • Microsoft Windows Server 2012 R2
  • Microsoft Windows 10
  • Microsoft Windows 11

Available architectures:

  • x86: Runs natively on x86 (32-bit) Windows, and via WOW64 on x64 and ARM64 Windows.
  • x64: Runs natively on x64 (64-bit) Windows, and via WOW64 on ARM64 Windows 11 or later.
  • ARM64: Runs natively on ARM64 (64-bit) Windows.
