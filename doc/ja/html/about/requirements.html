動作環境

Trea Term は以下の OS で動作します。

  • Microsoft Windows 95 (*1) (*2) (*3) (*4)
  • Microsoft Windows 98, Me
  • Microsoft Windows NT 4.0 SP6 (*2) (*4)
  • Microsoft Windows 2000, XP
  • Microsoft Windows 2000 Server
  • Microsoft Windows XP x64
  • Microsoft Windows Vista
  • Microsoft Windows Server 2003, 2003 R2
  • Microsoft Windows Server 2008, 2008 R2
  • Microsoft Windows 7
  • Fedora 13 with wine 1.3.4 (*5)

注: Windows 3.1 はサポートされていません。
(*1) Windows Sockets2.0 が必要です。
(*2) Internet Explorer 5 以上が必要です。
(*3) LogMeTT, TTL Editor は動きません。
(*4) Tahoma フォントのインストールを推奨します。
(*5) 端末内の日本語が化ける場合は、フォントの指定で「VLゴシック」を選び、文字セットに「Japanese」を選択して下さい。
