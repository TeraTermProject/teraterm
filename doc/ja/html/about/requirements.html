動作環境

Tera Term は以下の Windows で動作します。

  • Microsoft Windows 7
  • Microsoft Windows 8
  • Microsoft Windows Server 2012
  • Microsoft Windows 8.1
  • Microsoft Windows Server 2012 R2
  • Microsoft Windows 10
  • Microsoft Windows 11

Tera Term は以下の アーキテクチャ で動作します。

  • x86: x86 (32 ビット版) Windows 上でネイティブに動作し、x64 と ARM64 版 Windows では WOW64 経由で動作します。
  • x64: x64 (64 ビット版) Windows 上でネイティブに動作し、Windows 11 以降の ARM64 版 Windows ではWOW64 経由で動作します。
  • ARM64: ARM64 (64 ビット版) Windows 上でネイティブに動作します。
