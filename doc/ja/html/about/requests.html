要望集

ユーザーの方々からよくよせられる要望です。(つまり実現できていないものです。)
ヘルプ(MACRO ヘルプ、Keycode)を参照しても解決できないことがありましたら、問い合わせください。

将来対応するかもしれないもの

  • VT100の倍角文字、点滅文字
  • ECMA-48端末の低輝度文字、イタリック体、高速点滅文字、不可視文字、抹消線文字
  • VT3XXの文字(行)間罫線
  • ツール・バー、ステータス・バー
  • ダイアラー
  • キーボード設定用ユーティリティー
  • メニューやダイアログボックスのカスタマイズ
  • 文字列検索機能
  • TEK window の画面バッファ & スクロール・バー
  • TEK window の印刷の解像度を改善する。
  • ファイル転送時の転送速度、残り時間、棒グラフ表示[1]
  • ZMODEM の転送再開機能
  • [File] Print setup コマンド
  • スクロールバッファの内容、選択された文字をファイルに保存[2]
  • 画面/文字色の指定をカラーパレットで行う
  • 132 桁モードのとき小さいフォントに切り替える
  • SFTP(SSH File Transfer Protocol)
  • OpenSSHのProxyCommand

将来対応するかもしれないもの(MACRO 関係)

  • 配列変数[3]
  • "exec" コマンドでリダイレクション対応
  • バイナリデータを "wait" で待てるようにする。
  • 既に起動しているTera Termへのconnect

遠い遠い将来に対応するかもしれないもの

  • TEK41XX, 42XX 端末のエミュレーション
  • Televideo, Wyse 端末のエミュレーション
  • TN3270
  • 開いているシリアルポートを他のアプリケーションと同時共有可能にする。
  • 行入力モード、行入力のためのウィンドウを用意する。
  • Kermit の転送速度改善[4]
  • 可変幅(プロポーショナル)フォント
  • ANSI/AVATAR
  • ReGIS
  • Sixel(シクセル)
  • マクロで直接通信ポートを操作
  • xterm, DEC Locator mode以外のマウスプロトコル(NetTerm等)

注

  1. ^ Ver. 4.60でプログレスバーの表示に対応しました。
  2. ^ Tera Termのインストール先ディレクトリにある、screencapture.ttlが利用できるかもしれません。
  3. ^ Ver. 4.72で対応しました。
  4. ^ Ver. 4.73でLong Packetに対応しました。
