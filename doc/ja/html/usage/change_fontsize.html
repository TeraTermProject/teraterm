フォントサイズ変更プラグイン

Tera Term のメニューに [FontSize] メニューを追加する。

機能

TTXChangeFontSize プラグインを入れると、Tera Term のメニューに [FontSize] メニューを追加する事が出来ます。
[FontSize] メニュー内の項目が選択されると、そのサイズに Tera Term の VT ウィンドウのフォントサイズが変更されます。

使用方法

プラグインを入れただけでは [FontSize] メニューは表示されません。
設定ファイルにメニュー項目を登録する必要があります。

メニュー内の項目を変更したい場合は、TERATERM.INI に [Change Font Size] セクションを作成し、"FontSizeN= サイズ" の形式で指定して下さい。N は 1 からの連番です。
サイズにはフォントの文字セルまたは文字の高さを論理単位で指定します。正の値は文字セルの高さ、負の値は文字の高さとして扱われます。
メニューの項目数は最大で 20 です。

以下のメニュー IDが割り当てられています。
これらのメニュー IDを使用して、マクロのcallmenuコマンドやKEYBOARD.CNF設定から本機能を呼び出すことが出来ます。

TTXChangeFontSize メニューコマンド ID
[FontSize] Menu(1) - Menu(20) 58101 - 58120
[FontSize] Increase 58151
[FontSize] Decrease 58152

設定例

TERATERM.INI 
[Change Font Size]
FontSize1 = -15
FontSize2 = -19
KEYBOARD.CNF 
[User keys]
; Control + 1: Increase font size
User1=1026,3,58151
; Control + 2: Decrease font size
User2=1027,3,58152
