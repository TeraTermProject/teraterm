キーボード

キーボードに関するいくつかの設定は [Setup] Keyboard でできます。

キーボード設定ファイルを書きかえることで自由にキー設定を変更することができます。詳しくは、設定ファイル/キーボード設定 概要 を読んでください。

ショートカット・キー

VT window

ヒント: 「ユーザー・ショートカット・キー」

Alt+B
ブレーク信号を送出します。
Alt+C, Ctrl+Insert
マウスで選択した文字列をクリップボードへコピーします。 AutoTextCopy が有効な場合は、マウスで選択した時点でコピーされています。
Alt+D
セッションを複製します。（SSH, telnetのみ）
Alt+G
Cygwin接続を呼び出します。
Alt+I
切断メニューを呼び出します。
Alt+N
新しいホストまたはシリアルポートに接続します。
Alt+P
現在の画面または、マウスで選択された文字を印刷します。
Alt+Q または Alt+F4
Tera Term を終了します。
Alt+R
クリップボードの中の文字列をホストへ送出し、続いて改行コードも送出します。
Alt+T
Telnet の AYT (Are You There) コマンドを送出します。通常ホストはこれに呼応して何等かのメッセージを送り返してきます。Telnet 接続のときだけ使用できます。
Alt+V, Shift+Insert
クリップボードの中の文字列をホストへ送出します。 マウス右クリックでも同じです。 DisablePasteMouseMButton を off にしている場合は、マウス中ボタンクリックでも同じです。
Ctrl+Tab, Ctrl+Shift+Tab
Tera Termウィンドウを切り替えます。
Ctrl+Up/Down cursor
上下方向へ一行スクロールします。
Ctrl+PageUp/PageDown
上下方向へページ単位スクロールします。
Ctrl+Home
バッファの先頭へ移動します。
Ctrl+End
バッファの末尾へ移動します。
ScrollLock
Tera Termウィンドウのスクロールロック/ロック解除（トグル）
Alt+Enter
ウィンドウの最大化（トグル） タイトルバーが HideTitle で非表示になっていなければ、 Alt+Space でアクティブウィンドウのコンテキストメニューから Alt+X でも最大化できます。
F1 key
Num pad NumLockキー押下。
F2 key
Num pad '/' キー押下。
F3 key
Num pad '*' キー押下。
F4 key
Num pad '-' キー押下。

TEK window

Alt+C
マウスで選択された領域の画像をクリップボードへコピーします。
Alt+O
VT window に移動します。
Alt+P
画面全体または、マウスで選択された領域の画像を印刷します。
Alt+Q または Alt+F4
TEK window を閉じます。
Alt+R
VT window と同じ。
Alt+V
VT window と同じ。
Ctrl+Tab
VT window と同じ。

メタ・キー (Meta key)

[Setup] Keyboard で Meta key を選択すると、VTWindow で Alt キーをメタ・キーとして使うことができます。例えば Alt+A を押すと、ESC A を送出します。メタ・キーモードでは、Alt キーを使うショートカット・キー(Alt+C 等)は使用できなくなります。
