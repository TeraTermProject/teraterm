Receive file ([File] メニュー)

ファイルを受信します。

メニューを選択すると "Receive file" ダイアログが表示されます。
CTRLキーを押しながらをメニューを選択すると、 "Receive file" ダイアログをスキップして、受信ファイルの選択のみとなります。
この時、受信オプションは前回と同じものが使用されます。

この動作は 特別オプション で 変更することができます。

"Receive file" ダイアログボックス(受信前のファイル名指定)

Filename (drop file in this dialog)
受信するファイルのファイル名を入力または選択してください。
ダイアログにファイルをドロップすることでファイル名を入力することもできます。
Binary
ファイルの内容をそのまま受信します。
テキストモード(漢字コード、改行文字の変換、制御文字の出力抑止)は未実装のため、バイナリモード固定です。
Auto-stop wait time(sec)
受信データが無い状態が指定秒数続いた場合に、ファイル受信を終了します。
待ち時間のデフォルト値は5秒です。
0秒を指定した場合は、ファイル受信を終了せずデータを待ち続けます。

設定ファイルの特別オプション

  • ファイル受信ダイアログをスキップする

受信データのドロップ

シリアルポートから高ビットレートでデータを受信する場合、受信データをドロップする場合があります。
PCが高負荷で Tera Term の処理(UART チップからのデータ読み出し、ログ/ファイル出力、画面描画等)が追い付かない場合に発生しがちです。
Send file ([File] メニュー) 送信サイズも参照下さい。

"Receive file" ダイアログボックス(受信中)

Cancel
キャンセル ボタンを押すとファイル受信を中止します。
