[Setup] メニュー

変更した設定は自動的に保存されません。変更を恒久的に維持するには [Save setup...] メニューから手動で保存する必要があります。

Terminal...
端末エミュレーションの設定をします。
Window...
画面表示の設定をします。
Font...
Additional setting の フォント タブが開きます。
Keyboard...
Additional setting の キーボード タブが開きます。
Serial port...
シリアルポートの設定をします。
Proxy...
プロキシの設定をします。
SSH...
SSH 全般の設定をします。
SSH Authentication...
SSH 認証の設定をします。
SSH Forwarding...
SSH ポートフォワーディングの設定をします。
SSH Keygenerator...
SSH のキーペア認証で用いるキーの生成を行います。
TCP/IP...
Additional setting の TCP/IP タブが開きます。
General...
その他の設定をします。
Additional settings...
Additional setting が開きます。
Save setup...
現在の設定を Tera Term 設定ファイルに保存します。
Restore setup...
設定ファイルの内容にしたがって Tera Term を設定しなおします。
Setup directory...
読み込んでいる設定ファイルのパスを開きます。
Load key map...
キーボード設定ファイル の内容にしたがってキーボードを設定しなおします。
