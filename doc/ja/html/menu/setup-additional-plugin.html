Additional settings / "Plugin" タブ ([Setup] メニュー)

プラグインの許可/禁止を設定します。
プラグインフォルダのプラグイン、設定の一覧が表示されます。
パス
プラグインのパス
enable
右クリックで変更することができます。
設定は次の起動時に有効になります。
enable
プラグインをロードする。
disable
プラグインをロードしない。
unspecified
未設定(設定していない)。
未設定ファイルはロードされます。
loaded
プラグインがロードされているかどうかを示します。
loaded
プラグインはロードされている。
unloaded
プラグインはロードされていない。
order
ロードされているプラグインの優先順位です。
