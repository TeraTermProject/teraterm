Additional settings / "Plugin" タブ ([Setup] メニュー)

プラグインの許可(ロードする)/禁止(ロードしない)を設定します。

プラグインフォルダのプラグイン、設定の一覧が表示されます。

プラグインのロードは起動時に行われます。 起動後にプラグインをロードする、ロードを取り消すことはできません。 設定ファイルの再読み込みを行っても、プラグインが設定どおりに再ロード/ロードの取り消しが行われることはありません。

パス
プラグインのパス
enable
右クリックで変更することができます。
設定は次の起動時に有効になります。
enable
プラグインをロードする。
disable
プラグインをロードしない。
unspecified
enable/disable設定を行っていません。 設定を行っていない状態が初期状態です。
設定を行っていないプラグインはロードされます。
loaded
プラグインが現在ロードされているかどうかを示します。
loaded
プラグインはロードされている。
unloaded
プラグインはロードされていない。
order
ロードされているプラグインの優先順位です。
