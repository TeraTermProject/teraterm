エンコーディング

Tera Term 4.102より、Unicode (UTF-8, UTF-16BE, UTF-16LE) に対応しました。

従来はANSI CodePageとして処理していました。

1) サポートするエンコーディング

encoding support
従来どおり (ANSI CodePage) Support
UTF-8 (with BOM) Support
UTF-8 (without BOM) Support
UTF-16BE (without BOM) Not support
UTF-16LE (without BOM) Not support
UTF-16 (with BE BOM) Support
UTF-16 (with LE BOM) Support
UTF-16 (without BOM = UTF-16BE) Not support

2) エンコーディングの判定

Tera Termは次のように判定します。

BOMがある場合
BOMの種別からUTF-8, UTF-16BE, UTF-16LEを判定して読み込む
BOMがない場合
  • UTF-8として読み込む
  • UTF-8としてデコードできなかった場合はCP_ACP(4.102以前と同じ)として読み込む
