dirname

パス名からディレクトリ名を得る。(バージョン4.69以降) 

dirname <strvar> <path>

解説

パス名 <path> のディレクトリ名部分を <strvar> に格納する。
パス名は文字列として処理しており、実在している必要はない。

例

dirname dir 'c:\teraterm\test.txt'
; dir = "c:\teraterm"
dirname dir 'c:\teraterm\'
; dir = "c:"
dirname dir 'c:\teraterm'
; dir = "c:"
