intdim

整数配列型の変数を宣言する。 

intdim <array size>

解説

<array size>個の要素を持つ整数配列型の変数を宣言する。<array size>の最大値は65536。
配列内の各要素の初期値は0。

例

intdim fibonacci 20
fibonacci[0] = 0
fibonacci[1] = 1
for i 2 19
        fibonacci[i] = fibonacci[i-2] + fibonacci[i-1]
next

msg = ""
for i 0 19
        sprintf2 msg "%s%d, " msg fibonacci[i]
next

messagebox msg "result"
