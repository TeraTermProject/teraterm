listbox

リストボックスを開き、ユーザーに項目を選択させる。 

listbox <message> <title> <item1> [<item2>] [...] [<item10>]

パラメータ

文字列 <message>
リストボックスに表示されるメッセージ。
文字列 <title>
リストボックスのタイトル。
文字列 <itemN> (N=1-10)
リストボックスの選択項目。

返り値

システム変数 <result>
項目が選択された場合、0 - (N-1)のいずれかの値が格納される。
キャンセルされた場合、-1が格納される。

例

listbox '好きな食べ物を選んでください' 'あなたへの問い' 'バナナ' 'りんご' 'みかん'
sprintf2 var "%d" result
messagebox var "result"
; result
;   -1: Cancel
;    0: Banana
;    1: Apple
;    2: Orange
