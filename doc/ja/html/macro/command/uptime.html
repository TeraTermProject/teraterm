uptime

OSの稼働時間を取得する。(バージョン4.81以降)

1) 形式1

uptime <intvar>

2) 形式2

uptime <secvar> <msecvar>

説明

形式1

Windows OSの稼働時間（ミリ秒）を <intvar> に格納する。

※稼働時間が49日を経過すると、0に戻る。

形式2

GetTickCount64() を使ってWindows OSの稼働時間を取得し、下記の値を格納する。

<secvar>
Windows OSの稼働時間（秒単位）(GetTickCount64() / 1000)
<msecvar>
稼働時間の1秒未満の部分（ミリ秒単位）(GetTickCount64() % 1000)

<secvar> と <msecvar> はともに整数型変数でなければならない。

例

; 形式1: Windowsの稼働時間を取得する例
uptime var
day = var / 86400000
var = var % 86400000
hour = var / 3600000
var = var % 3600000
min = var / 60000
var = var % 60000
sec = var / 1000

sprintf2 str "System Up Time: %d Days, %d Hours, %d Minutes, %d Seconds" day hour min sec
messagebox str "uptime"

uptime varstart

; do something...

uptime varend
diff = varend - varstart
sprintf2 str "MACRO running time: %d msec" diff
messagebox str "running time"

; 形式2: 稼働時間を秒とミリ秒で取得する例
uptime sec msec
sprintf2 str "System Up Time: %d.%03d seconds" sec msec
messagebox str "uptime"
