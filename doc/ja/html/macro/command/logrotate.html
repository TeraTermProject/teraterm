logrotate

ログをローテートする。 

logrotate 'size' <num>
logrotate 'halt'

パラメータ

'size' <num>
ログのサイズが<num>バイトを超えていれば、ローテーションを行う。
<num>は128以上であること。
'halt'
ローテーションを停止する。

例

getspecialfolder str "Desktop"
changedir str
logopen 'teraterm_sample.log' 0 0
; Start log rotation
logrotate 'size' 2048
pause 600
; Stop log rotation
logrotate 'halt'
pause 600
logclose
