strtrim

前後の文字を削除する。

形式

strtrim <strvar> <trimchars> [<special>]

解説

文字列変数 <strvar>の先頭と末尾から、<trimchars> で指定された文字をすべて削除する。

<special> が 1 の場合は、<separator> に含まれる以下の文字列は特殊文字として扱われる。
<special> が指定されない場合は 1 が使用される。

入力 出力
\\ "\"
\n 改行
\t タブ

例

src='   '#9' #include       '#9' '
strtrim src ' \t'
messagebox src 'result'          ; '#include '

src='         #include          '
strtrim src ' '
messagebox src 'result'          ; '#include '

src='*;|@123***456@|;'
strtrim src '*;|@'
messagebox src 'result'          ; '123***456'

src='*;|@123***456@|;'
strtrim src '*;|@123456'
messagebox src 'result'          ; ''

src='#*;|@123***456@|;'
strtrim src '*;|@'
messagebox src 'result'          ; '#*;|@123***456'

src='*;|@123***456@|;#'
strtrim src '*;|@'
messagebox src 'result'          ; '123***456@|;#'
