strtrim

前後の文字を削除する。

形式

strtrim <strvar> <trimchars>

解説

文字列変数 <strvar>の先頭と末尾から、<trimchars> で指定された文字をすべて削除する。

例

src='         #include          '
strtrim src ' '
messagebox src 'result'          ; '#include '

src='*;|@123***456@|;'
strtrim src '*;|@'
messagebox src 'result'          ; '123***456'

src='*;|@123***456@|;'
strtrim src '*;|@123456'
messagebox src 'result'          ; ''

src='#*;|@123***456@|;'
strtrim src '*;|@'
messagebox src 'result'          ; '#*;|@123***456'

src='*;|@123***456@|;#'
strtrim src '*;|@'
messagebox src 'result'          ; '123***456@|;#'

参照

strinsert
strremove
