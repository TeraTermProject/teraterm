bringupbox

statusbox コマンドによって開かれたステータスダイアログボックスを最前面に表示する。 

bringupbox

パラメータ

なし。

返り値

なし。

例

step = 0
while 1
  call show_step
  sendln 'echo do something'
  wait '$'
  pause 2

  bringupbox

endwhile
end

:show_step
step = step + 1
gettime timestr "%Y/%m/%d-%H:%M:%S"
sprintf2 s "counter %d" step
statusbox timestr s
return
