messagebox

ダイアログボックスを開き、ユーザーにメッセージを知らせる。 

messagebox <message> <title> [<special>(廃止)]

解説

文字列 <message> はダイアログボックスに表示されるメッセージ、文字列 <title> はダイアログボックスのタイトルを表す。

<special> は廃止されました。エラーにはなりませんが、無効です。

例

messagebox ErrorMessage 'Error'

title="5行表示"
message="1行目\n2行目\n3行目\n4行目\n5行目\n"
strspecial message
messagebox message title
