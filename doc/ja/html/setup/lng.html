言語ファイル

Tera Term のメニューやダイアログには、英語のメッセージが割り当てられています。
言語ファイルを作成することにより、メッセージをローカライズすることができます。
また、このファイルにはダイアログのフォントとフォントサイズも指定することができます。

インストーラは、lang フォルダの中に日本語・ドイツ語・フランス語・ロシア語・韓国語・中国語・スペイン語・タミル語の言語ファイルをコピーします。
また、インストーラは指定された言語が使用されるように、TERATERM.INI の UILanguageFile に言語ファイル名を指定します。

フォントの指定

インストーラは、フォントだけを指定するテンプレートとして Default.lng をコピーします。
フォントの指定は以下のようなフォーマットで行います。 

; 大きいフォント
DLG_SYSTEM_FONT=System,12,0
; 小さいフォント
DLG_TAHOMA_FONT=Tahoma,8,0

一つ目にはフォント名を指定します。

二つ目にはフォントサイズを数字で指定します。

三つ目にはフォントのキャラクタセットを数字で指定します。 

0   ANSI
1   既定の文字セット
2   シンボル(記号)
77  Mac
128 日本語(Shift_JIS)
129 韓国語(EUC-KR)
130 韓国語(Johab)
134 簡体字中国語(GB2312)
136 繁体字中国語(Big5)
161 ギリシャ語
162 トルコ語
163 ベトナム語
177 ヘブライ語
178 アラビア語
186 バルト言語
204 キリル言語
222 タイ語
238 中央ヨーロッパ言語
255 OEM/DOS

詳しくは msdn ライブラリの LOGFONT 構造体の lfCharset の説明を参照してください。
