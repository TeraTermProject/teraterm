Tera Term 設定ファイル

起動時または、[Setup] Restore setupを実行したとき、設定ファイルの内容にしたがい Tera Term が初期化されます。
コマンドラインで設定ファイル名を指定することができます。ファイル名の拡張子を省略した場合は、".INI"であるとみなされます。

Tera Term 設定ファイルの全項目一覧

変更した設定は設定ファイルに自動的に保存されません。現在の設定を設定ファイルに保存するには、[Setup] Save setupを実行してください。

Tera Term 設定ファイルの特別オプション

以下の設定をするには、ユーザーは Tera Term 設定ファイルを直接編集しなければなりません。

ターミナルエミュレーション

  • バック・ラップ (back wrap)
  • BEL 文字受信時の動作
  • C1 制御文字($80-$9F)を使用しない
  • C1 制御文字($80-$9F)を送信に使用する
  • JIS 漢字送出の時、正しくない "kanji-out" シーケンス ^[(H を使用可能にする
  • 2バイト文字(漢字)の1バイト目と2バイト目の間に制御文字等が入った場合、1バイト目を無効にする
  • 受信した文字をすべて表示するようにする
  • ステータスラインを使用不可にする
  • TEK GIN report のマウスコードの値を設定する
  • 行末のタブ表示を VT 端末互換にする
  • 画面クリア時にカレントバッファをスクロールアウトさせるかどうか設定する
  • UnicodeからDEC Special Graphicsへのマッピングを設定する
  • ファンクションキー等のデフォルトの動作
  • AltキーでMSBをセットした文字を送信する
  • 代替画面バッファを無効にする
  • 端末固有IDを設定する
  • 端末固有IDを変更可能にする
  • タブストップ位置の変更を禁止する
  • ベル過使用時の一時無効化
  • ISO/IEC 2022のシフト機能による文字セットの切り替えを禁止する
  • 7bit JISカタカナ(専用)端末として使用する
  • 不正なUTF-8文字をCP932としてデコードする

ウィンドウ

  • 自動的な文字列コピーをしない
  • ダブルクリックによる単語選択の区切り文字
  • ポップアップメニューを使用不可にする
  • "Show menu bar" コマンドを隠す
  • 言語モードの選択を不可にする
  • スクロールバッファーの最大行数
  • 点滅しないカーソル
  • 一度にスクロールする最大の行数を設定する
  • マウスでウィンドウを選択したときの文字の選択を禁止する
  • タイトルの形式
  • 白色および黒色属性の文字の実際の色
  • 文字(行)間のすきまを調整する
  • [Window] メニューを隠す
  • ウィンドウ最大化時のスクロール方法
  • フォーカスが外れた時にもカーソルを表示する
  • ウィンドウの初期位置を固定する
  • ウィンドウ位置を保存する
  • ウィンドウアイコンを変更する
  • マウスでの文字の選択の開始を遅らせる
  • ウィンドウの角を丸くしない

通信

  • シリアルポート番号の最大値
  • シリアルポートのバッファをクリアしない
  • TCP/IP 回線切断の確認をしない
  • Telnet のバイナリオプション
  • Telnet のエコーオプション
  • Telnet のログ
  • Telnet 用ポート番号 / 非 telnet 接続用端末設定
  • Telnet コマンド自動検出の無効化
  • 行単位送信モードの無効化
  • シリアルポートでブレーク信号を送出する時間の長さ
  • シリアルポートの切断・再接続を検出する
  • サーバに通知する端末速度を変更する

ファイル転送

  • 受信したファイルの名前を自動的に変更して上書きを避ける
  • B-Plus 受信の自動起動
  • B-Plus ですべての制御文字をエスケープする
  • B-Plus のログ
  • Kermit のログ
  • Quick-VAN のログ
  • Quick-VAN のウィンドウサイズ
  • XMODEM のログ
  • XMODEM の受信コマンド
  • XMODEM のタイムアウト
  • YMODEM のタイムアウト
  • ZMODEM 受信/送信の自動起動
  • ZMODEM 送信の設定
  • ZMODEM ですべての制御文字をエスケープする
  • ZMODEM のログ
  • ZMODEM の受信コマンド
  • ZMODEM のタイムアウト
  • 送信ダイアログのファイル名フィルタ
  • ドラッグ&ドロップによるファイル送信の確認
  • ファイル送信時の転送状況ダイアログを表示しない
  • シリアル接続における高速なファイル転送

印刷

  • エスケープシーケンスによる印刷で直接ポートに出力

  • エスケープシーケンスによる印刷時の改ページを禁止する
  • プリンタフォントの指定
  • ページ余白の指定
  • 印刷の倍率

  • ブロードキャストコマンドの履歴の保存数

  • 起動時の "New Connection" ダイアログの表示
  • マウスの中ボタンによる貼り付けを有効にする
