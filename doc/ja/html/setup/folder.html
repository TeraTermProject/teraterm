ポータブル版について

  • 環境(システム設定、個人設定)に左右されず使用することができる
    • 次のような用途を想定
      • USBメモリに入れて、そこから起動する
      • インストールせずに使用する (コピーするだけで使えるようになる)
  • 実行する ttermpro.exe と同じフォルダに portable.ini ファイルを置くことでポータブル版として動作
  • portable.ini の中身はなんでもよい(サイズ0でよい)

インストール版の使用するフォルダ,ファイルについて

Windows Vista以降 の場合のフォルダ例

  • 設定ファイル
    • C:\Users\[username]\AppData\Roaming\teraterm5
  • 動作ログ、ダンプ
    • C:\Users\[username]\AppData\Local\teraterm5

Windows 2000, XP の場合のフォルダ例

  • 設定ファイル
    • C:\Documents and Settings\[username]\AppData\Roaming\teraterm5
  • 動作ログ、ダンプ(未定義)
    • C:\Documents and Settings\[username]\Application Data\teraterm5

ファイルを指定した場合

コマンドラインでファイルをフルパスで指定した場合は、それを優先して使用する

  • TERATERM.INI
  • KEYBOARD.CNF
  • broadcast.log

設定ファイルがない場合

ttermpro.exeは次のように動作する

  • 設定ファイルを置くフォルダを作成する
  • ttermpro.exeのフォルダにある設定ファイルを "設定ファイルを置くフォルダ" にコピーする
