Window (TEK window [Setup] メニュー)

"Window setup" ダイアログボックス

Title
ウィンドウのタイトル。
Cursor shape
カーソルの形。Block, Vertical line, Horizontal line の3つから選べます。
Hide title bar
タイトルバーとメニューバーを消します。ウィンドウを移動させるには Alt キーを押しながら右マウスボタンでウィンドウをドラッグしてください。ウインドウを最小化させるには Alt キーを押しながら左マウスボタンを押してください。Ctrl キーを押しながら左マウスボタンを押すとポップアップメニューが現れます。
Hide menu bar
メニューバーを消します。Ctrl キーを押しながら左マウスボタンを押すとポップアップメニューが現れます。
Color emulation
文字やグラフィックの色を指定するエスケープシーケンスを有効にします。このオプションを選択すると、選択しない場合に比べメモリ消費量が増加します。必要のない場合は選択しないでください。
Color
Text
文字の色
Background
背景の色
Reverse
文字の色と背景の色を交換します。
