[Window] メニュー (TEK window)

実行中の Tera Term ウィンドウをこのメニューのリストから選び、画面の一番前に表示させます。最大で9個のウィンドウがリストに表示されます。実行中のウィンドウが9個より多く、リストにないウィンドウを選択したい場合は、このメニューにある Window コマンドを使用してください。

Window...
実行中の Tera Term ウィンドウを画面の一番前に表示させたり閉じたりします。
Entering Single Window Mode
シングルウィンドウモードに入ります。
シングルウィンドウモードでは、表示される Tera Term ウィンドウは1つのみになります。
Tera Term ウィンドウを切り替えると、切り替え前と同じ位置に切り替え後のTera Term ウィンドウが表示されます。
Tera Term ウィンドウの切り替え方法は以下のとおりです。
  1. メニューバーの "<" 又は ">" をクリックする
    2. シングルウィンドウモードでのみ、メニューバーに "<" と ">" が表示されます。
    "<" : 前のウィンドウに切り替えます。
    ">" : 次のウィンドウに切り替えます。
  2. CTRL+SHIFT+TAB 又は CTRL+TAB を押下する
    3. "CTRL+SHIFT+TAB" : 前のウィンドウに切り替えます。
    "CTRL+TAB" : 次のウィンドウに切り替えます。
  3. [Window] メニューのウィンドウリスト又はメニュー内の Window コマンドを使用する
Exiting Single Window Mode
シングルウィンドウモードを終了します。
