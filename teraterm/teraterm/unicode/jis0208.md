# JIS0208

- 元データ
  - http://www.unicode.org/Public/MAPPINGS/OBSOLETE/EASTASIA/JIS/JIS0208.TXT

コードポイントU+0080以上で、JIS0208の文字を全て2cell(全角)として扱う
iniファイルの一部を作成する。

iniファイルはUTF-8で作成される。

# iniファイルの作り方

```
wget http://www.unicode.org/Public/MAPPINGS/OBSOLETE/EASTASIA/JIS/JIS0208.TXT
perl jis0208.pl jis0208.ini
```
